# Code of Conduct

## Our Pledge
We pledge to make participation in our project a harassment-free experience.

## Our Standards
Examples of acceptable behavior include respectful communication.

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing standards.

## Scope
This Code of Conduct applies to all project spaces.

## Enforcement
Instances of abusive behavior may be reported to the maintainers.

## Enforcement Guidelines
Consequences range from warnings to permanent bans.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
